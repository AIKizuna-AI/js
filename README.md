# 简介

《每天一个JS程序》系列链接的源码。

欢迎大家三连（star、watch、fork）。

# 使用方法

通过下面的目录找到你需要的JS程序

如果你想使用，点击压缩版，复制所有代码，在收藏夹添加标签，在网址处输入`javascript:`，粘贴上代码。

如果你想研究源码，点击注释版。
